    NIA-Server

    A Minecraft server based on BDS

    About UsDeploy Server

    Cross-Platform Server

    A server that can be played on all platforms! Play anytime, anywhere!

    Bedrock Edition Server

    The server is developed based on the official BDS, supporting play on platforms where Minecraft Bedrock (Bedrock Edition) can be downloaded, such as Android, iOS, Windows, etc.

    Update with Official Version

    Thanks to the features of Bedrock Edition, the server updates with the official version, providing you with the latest gaming experience at any time.

    Vanilla Survival Content Expansion

    We have made some modifications to the gameplay using official Add-ons based on vanilla survival, making your survival experience richer!

    New Challenges

    You can breathe underwater, but suffocate in the air! Are you ready for such a challenge?

    Creature Modification

    To match the expanded game content, we have modified the behavior of some creatures.

    Land Claim System

    Fully protect your territory, so it is no longer disturbed by the outside world.

    Trading Market

    To facilitate player transactions, we provide a trading market, making it easier for you to trade.

    All Add-ons and Related Components are Open Source

    All Add-ons and related components developed by the server are open source, allowing you to set up your own server according to your needs!

    Sync Server New Features

    The open-source Add-ons and related components will sync with the server's new features and update with the official version, providing you with the latest gaming experience.

    NIAHttpBOT

    Based on HTTP, it implements operations such as reading and writing files, giving the original script-api more capabilities.

    Comprehensive Development Documentation

    There is relatively comprehensive development documentation here to help you quickly get started and understand how to use our open-source projects.

    Add-ons Function Separation

    Add-ons are divided into NiaServer-Core and NiaServer-ExtraFeatures, allowing you to selectively use them according to your needs.
